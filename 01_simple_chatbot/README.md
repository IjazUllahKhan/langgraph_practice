# Simple Chatbot

A simple chatbot implementation to learn LangGraph fundamentals.

## Overview

This project demonstrates basic LangGraph concepts including:

- Graph-based conversation flow
- State management
- Node composition
- Basic chat interactions

## Files

- `simple_chatbot.ipynb` - Jupyter notebook with chatbot implementation

## Running the Chatbot

1. Start Jupyter Notebook:

```bash
jupyter notebook
```

2. Open `simple_chatbot.ipynb`

3. Run the cells to execute the chatbot

## Requirements

- langchain
- langgraph
- jupyter
- python-dotenv (for environment variables)

## Installation

```bash
pip install langchain langgraph jupyter python-dotenv
```

## Notes

- Remember to set up your API keys in a `.env` file for any LLM providers
- This is a practice project to understand LangGraph concepts
